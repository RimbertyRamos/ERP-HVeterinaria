VetCare UAGRM: Sistema Modular 3NF

Estructura del Proyecto: Este modelo representa el diseño lógico del sistema VetCare UAGRM en su versión optimizada. Se basa en una arquitectura de base de datos relacional normalizada (3NF), diseñada para gestionar de forma eficiente el ciclo de vida del paciente veterinario, desde su registro inicial hasta la facturación final.
Módulos Operativos: El esquema integra cinco pilares fundamentales: Recepción y Turnos (Fichaje), Gestión Clínica (Historia Clínica y Registro SOAP), Farmacia Integral (Control de Stock, Kardex y Compras a Proveedores), Laboratorio Clínico (Órdenes y Resultados) y Punto de Venta (POS) para el cobro centralizado de servicios y productos.
Gestión y Control: El sistema está diseñado bajo un modelo de Control de Acceso (RBAC), donde el perfil ADMIN ejerce una supervisión transversal, garantizando la integridad de los datos en todos los niveles operativos y financieros del hospital veterinario.
