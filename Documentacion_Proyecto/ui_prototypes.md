# Prototipos de Interfaz: Sistema Veterinario Integral

Este documento presenta la propuesta visual para asegurar que el sistema sea intuitivo y profesional para cada rol.

## 1. Dashboard (Administrador / Recepción)
El panel principal ofrece una vista rápida del estado del hospital.

![Dashboard Mockup](file:///C:/Users/Asus/.gemini/antigravity/brain/fc4334f3-d155-402c-93f6-b01d1dc0106f/dashboard_mockup.png)

*   **KPIs**: Consultas activas, ingresos del día, alertas de stock bajo.
*   **Fichado**: Visibilidad rápida de qué consultorios están ocupados.

## 2. Módulo de Consulta (Doctor)
Optimizado para que el doctor registre la atención sin distracciones, integrando inventario y laboratorio.

![Consultation Mockup](file:///C:/Users/Asus/.gemini/antigravity/brain/fc4334f3-d155-402c-93f6-b01d1dc0106f/consultation_mockup.png)

*   **Historia Clínica**: Acceso rápido a visitas anteriores mientras escribe el diagnóstico actual.
*   **Gestión de Insumos**: Buscador lateral para añadir jeringas, medicamentos o gasas, descontando stock en tiempo real.
*   **Botón de Cobro**: Al "Finalizar", envía automáticamente el detalle a Caja.

## 3. Módulo de Caja (Cajero)
Enfocado en la velocidad y precisión del cobro.

![Cashier Mockup](file:///C:/Users/Asus/.gemini/antigravity/brain/fc4334f3-d155-402c-93f6-b01d1dc0106f/cashier_mockup.png)

*   **Cola de Cobro**: Listado automático de pacientes que terminaron su consulta o análisis.
*   **Detalle Transparente**: Desglose claro de honorarios médicos + materiales + farmacia.
*   **Seguridad Contable**: Botones de anulación protegidos por rol de administrador.

---

## 4. Estándares de Diseño (Design System)
*   **Modo Oscuro (Dark Mode)**: Para reducir el cansancio visual del personal médico.
*   **Colores**: Verde esmeralda (salud/dinero), Azul (profesionalismo), Rojo (alertas).
*   **Tipografía**: Inter o Roboto para máxima legibilidad.
